## 🌟 ما هو المراقب الذكي؟

**المراقب الذكي (Builder Daemon)** هو تطبيق سطح مكتب وهاتف ذكي (أندرويد) يُحدث ثورة في طريقة تفاعلك مع نماذج اللغة الكبيرة (LLMs) مثل ChatGPT، Gemini، وDeepSeek.  
عندما يقدم لك مساعد ذكي ردًا يحتوي على عدة ملفات برمجية، نصوص، أو تقارير، فإنك عادةً تضطر إلى نسخ كل كتلة وحفظها يدويًا.  
**المراقب الذكي** يقوم بأتمتة هذه العملية بالكامل: بمجرد نسخ الرد إلى الحافظة، تتعرف الأداة على **التوجيهات** الخاصة (مثل `@builder:file src/main.py`) وتقوم بإنشاء الملفات في المسارات المحددة فورًا وبصمت.

---

## 🎯 المشكلة التي يحلها

في سير العمل المعتاد، عندما يقدم مساعد ذكي (مثل ChatGPT) ردًا يحتوي على عدة ملفات، يضطر المستخدم لنسخ كل كتلة كود وحفظها يدويًا في ملف منفصل.  
هذه العملية تستغرق وقتًا طويلاً، خاصة مع المشاريع الكبيرة التي تحتوي على عشرات الملفات.

**المراقب الذكي** يحل هذه المشكلة عبر أتمتة كاملة:

-   **نسخة واحدة** من الرد كافية.
-   **تتعرف الأداة تلقائيًا** على التوجيهات (مثل `@builder:file src/main.py`).
-   **تنشئ الملفات والمجلدات** في هيكل منظم.
-   **تدعم الكتابة والإلحاق والبادئات المتعددة والمسارات المخصصة**.

**النتيجة:** توفير ساعات من العمل اليدوي، والقضاء على الأخطاء الناتجة عن النسخ واللصق المتكرر.

## ✨ الميزات الرئيسية

### 🧠 محرك أوامر ذكي وقابل للتوسع
-   يدعم توجيهات `@builder:file`، `@executor`، `@treedoc`.
-   **أكثر من 15 أمرًا** لمعالجة الملفات: `mkdir`، `move`، `rename`، `copy-safe`، `delete`، `scan`، `report`...
-   **دعم JSON** لتمرير المعاملات المعقدة بأمان.
-   **دعم `--dry-run`** لمعاينة الأوامر قبل التنفيذ.

### 🛡️ طبقات أمان متعددة
-   **نسخ احتياطي تلقائي** قبل أي عملية تعديل (نقل، حذف، إعادة تسمية).
-   **منع الكتابة فوق الملفات** تلقائيًا مع إعادة تسمية ذكية.
-   **تأكيدات وحماية** ضد الحذف غير المقصود.
-   **نظام اختبار ذاتي** `@executor:selftest` لفحص المحرك بالكامل في بيئة آمنة.

### 📋 مراقبة الحافظة في الخلفية (أندرويد)
-   **ثلاثة مسارات** لضمان العمل على جميع إصدارات أندرويد:
    1.  **لوحة مفاتيح ذكية (IME)** للأجهزة الحديثة (Android 10+).
    2.  **فقاعة ذهبية عائمة** كمركز تحكم مصغر مع معالجة فورية.
    3.  **خدمة إمكانية الوصول** للأجهزة القديمة (Android 9-).

### 📊 تقارير تفاعلية متعددة الصيغ
-   توليد تقارير شجرية بصيغ: `TXT`، `JSON`، `HTML` تفاعلي، `CSV`.
-   **تصدير ونسخ** سجل الأحداث كاملاً مع إمكانية التحرير والفلترة.
-   **معاينة** الملفات والمجلدات وفتحها فورًا.

### 🌐 تكامل مع أي مساعد ذكي
-   **موجّه نظام (System Prompt)** جاهز للنسخ، يشرح لأي مساعد خارجي كيفية إصدار أوامر `@executor`.
-   يعمل مع ChatGPT، Gemini، DeepSeek، وأي نموذج آخر.

## 📦 محتويات المشروع

هذا المستودع يحتوي على كل ما تحتاجه لتشغيل التطبيق محليًا:

-   **تطبيق أندرويد كامل** (كود المصدر).
-   **ملف `.env.example`** لتكوين مفتاح Gemini API.
-   **ملف `test_scenarios.json`** يحتوي على سيناريوهات الاختبار الذاتي.

## 🚀 تشغيل المشروع محليًا

**المتطلبات الأساسية:** [Android Studio](https://developer.android.com/studio)

1.  افتح **Android Studio**.
2.  اختر **Open** وحدد المجلد الذي يحتوي على هذا المشروع.
3.  اسمح لـ Android Studio بإصلاح أي تعارضات أثناء استيراد المشروع.
4.  أنشئ ملفًا باسم `.env` في مجلد المشروع، وأضف إليه مفتاح Gemini API الخاص بك:
    ```
    GEMINI_API_KEY=your_api_key_here
    ```
    (انظر الملف `.env.example` للمثال).
5.  احذف هذا السطر من ملف `app/build.gradle.kts`:
    ```
    signingConfig = signingConfigs.getByName("debugConfig")
    ```
6.  شغّل التطبيق على محاكي أو جهاز حقيقي.

## ⌨️ الأوامر المدعومة (توجيهات `@executor`)

### 📁 أوامر إدارة الملفات والمجلدات
| الأمر | الوصف | مثال |
| :--- | :--- | :--- |
| `mkdir` | إنشاء مجلد | `@executor:mkdir --path="/NewFolder"` |
| `move` | نقل ملف/مجلد | `@executor:move --path="file.txt" --dest="Folder/"` |
| `rename` | إعادة تسمية | `@executor:rename --path="old.txt" --newName="new.txt"` |
| `copy-safe` | نسخ آمن | `@executor:copy-safe --path="file.txt" --dest="backup/"` |
| `delete` | حذف آمن | `@executor:delete --path="file.txt"` |
| `open` | فتح ملف/مجلد/رابط | `@executor:open --target="https://example.com"` |

### 🔍 أوامر المسح والتحليل
| الأمر | الوصف | مثال |
| :--- | :--- | :--- |
| `scan` | مسح مجلد وإرجاع قائمة | `@executor:scan --path="/Downloads" --format=json` |
| `report` | إنشاء تقرير | `@executor:report --path="/Project" --format=html` |

### 🧪 أوامر النظام
| الأمر | الوصف | مثال |
| :--- | :--- | :--- |
| `selftest` | تشغيل الاختبار الذاتي | `@executor:selftest` |

**ملاحظة:** يمكن استخدام صيغة JSON بدلاً من `--param` لتمرير المعاملات:
```json
@executor:move
{
  "path": "/file.txt",
  "dest": "/Folder/",
  "overwrite": false
}
```
